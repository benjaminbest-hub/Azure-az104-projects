# Project 02 - Automated 3-Tier Web Application Deployment

## Overview
This project demonstrates deploying a .NET 10 Web API to Azure App Service using Azure SQL Database and Managed Identity authentication.

## Azure Services Used
- Azure App Service
- Azure SQL Database
- Azure Managed Identity
- Azure Resource Group
- Azure App Service Plan

## Features
- REST API
- Entity Framework Core
- SQL Database
- Managed Identity Authentication
- Health Check Endpoint (/health)

## Lessons Learned
- Deploying .NET applications to Azure
- Configuring Managed Identity
- Using Entity Framework Core Migrations
- Troubleshooting App Service startup issues
