# Deployment Guide

## Objective

Deploy an enterprise Disaster Recovery solution using Azure Backup and Azure Site Recovery.

---

## Resources

- Recovery Services Vault
- Virtual Machine
- Azure Backup
- Azure Site Recovery

---

## Deployment Steps

1. Create Resource Group
2. Create Recovery Services Vault
3. Deploy Production VM
4. Enable Azure Backup
5. Create Initial Recovery Point
6. Configure Azure Site Recovery
7. Enable Replication
8. Validate Replication Health
9. Perform Test Failover
10. Validate Recovery
