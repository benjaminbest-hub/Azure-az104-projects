# Lessons Learned

## Azure Backup

Azure Backup protects data and allows recovery after failures.

---

## Azure Site Recovery

Azure Site Recovery keeps workloads available during disasters through continuous replication.

---

## Recovery Point Objective (RPO)

The maximum acceptable amount of data loss measured in time.

---

## Recovery Time Objective (RTO)

The target time required to restore business operations.

---

## Enterprise Best Practices

- Test failovers regularly
- Use Geo-Redundant Storage
- Monitor replication health
- Protect production workloads
- Document recovery procedures
